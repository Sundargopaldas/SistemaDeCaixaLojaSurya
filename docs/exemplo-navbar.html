Exemplo de NavBar Padronizada

Estrutura Básica

Esta página demonstra a estrutura básica de uma página com a navbar padronizada do PDV-Surya.

Observe os seguintes pontos importantes:

  • Link para navbar.css no head
  • Estrutura HTML da navbar com todos os links padrão
  • Espaçamento superior para acomodar a navbar fixa (margin-top)
  • Script navbar.js no final do body

Classes CSS Importantes

  • .nav-menu - Contêiner principal da navbar
  • .active - Classe para destacar o link da página atual
  • .user-menu - Menu do usuário à direita da navbar
