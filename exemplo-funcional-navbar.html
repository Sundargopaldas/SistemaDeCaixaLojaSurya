Exemplo Funcional: Navbar Padronizada

Esta página demonstra a implementação correta da navbar padronizada para o sistema PDV-Surya.

Como Funciona

Esta navbar utiliza:

  • Posicionamento fixo no topo da página
  • Estilo visual consistente com o dashboard
  • Destacamento automático da página atual
  • Menu dropdown para o usuário

Implementação em 4 Etapas

IMPORTANTE: Siga esta ordem exata para que a navbar funcione corretamente!

1. Adicionar CSS da Navbar

Adicione o seguinte link no <head> da sua página, antes de qualquer outro CSS:

<link rel="stylesheet" href="css/navbar.css">

2. Estrutura HTML da Navbar

Adicione esta estrutura no início do <body>:

<div class="nav-menu"> <a href="index.html">Início</a> <a href="caixa.html">Caixa</a> <a href="produtos.html">Produtos</a> <a href="produtoLista.html">Lista de Produtos</a> <a href="historico.html">Histórico</a> <a href="despesas.html">Despesas</a> <a href="contasPagar.html">Contas a Pagar</a> <a href="listaDeClientes.html">Clientes</a> <a href="listaDeFornecedores.html">Fornecedores</a> <a href="orcamentos.html">Orçamentos</a> <a href="dashboard.html">Dashboard</a> <a href="relatorios.html">Relatórios</a> <div class="user-menu"> <div class="user-icon">U</div> <div class="dropdown-content"> <a href="#" id="botao-sair">Sair</a> </div> </div> </div>

3. Adicionar Margem ao Conteúdo Principal

Para evitar que o conteúdo fique escondido sob a navbar, adicione uma margem superior:

<div class="seu-container-principal" style="margin-top: 70px;"> </div>

4. Adicionar JavaScript da Navbar

No final do <body>, adicione:

<script src="js/navbar.js"></script>

Verificação da Implementação

Dica: Use o script de diagnóstico para verificar a implementação em todas as páginas:
node scripts/diagnostico-navbar.js

Checklist Manual

  • A navbar está fixa no topo da página?
  • A página atual está destacada na navbar?
  • Ao passar o mouse sobre os links, eles mudam de cor?
  • O menu dropdown do usuário abre ao clicar no ícone?
  • A navbar se adapta corretamente em telas menores?
